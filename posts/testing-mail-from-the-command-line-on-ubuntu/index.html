Testing Mail From the Command Line on Ubuntu

September 19, 2012 One-minute read
Command Line • Ubuntu

Install mailutils

sudo apt-get install mailutils

Send some test piped into the mail command

echo "testing" | mail -s Test user@emailhost.com
