Google source code hosting

  1 minutes  |    85 words

If you don’t want to host your own sourcecode on your own Subversion server like I was doing, check this out.

It is Google’s code hosting for subversion.  You are given 100 MB of space and can apply a specific open source licensing rule to your source.

Here are the currently allowed licenses.

Apache License 2.0 Artistic License/GPLv2 GNU General Public License 2.0 GNU Lesser Public License MIT License Mozilla Public License 1.1 New BSD License
